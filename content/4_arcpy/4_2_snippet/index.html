Clip a layer

Use the clip tool to clip the denver_schools feature class using the five_points boundary.

Copy Python Snippet

  • In ArcMap, from the Geoprocessing menu, choose Results.
  • In the Results window, expand Current Session.
  • Right-click Clip, and then choose Copy As Python Snippet (as shown).


  • Open Python Window
    Go to Geoprocessing -> Python



    or Click the Python Window button on the Toolbar
  • In the Python window, right-click and choose Paste.
