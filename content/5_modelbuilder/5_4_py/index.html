Tools or workflow built with ModelBuilder can be exported to Python Script.

Export the iteration workflow we built in ModelBuilder to Python Script.
Open the exported Python script with IDLE, and read the code.

If you try to run the python script, you will get an error. That's because iterator from ModelBuilder is native to ModelBuilder and can not be interpreted by Python.

In the next section, we will write the iteration in Python to make the exported script work.
