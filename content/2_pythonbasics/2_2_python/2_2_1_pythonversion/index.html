Python 2 or Python 3?

Python 2.x

Last release is version 2.7 in 2010.

Python 3.x

Pthon 3.0 was released in 2008. After that, 3.x is under active development and has the latest releas of version 3.4 in 2014.

Which version should I use?

3.x downsides: slightly worse library support; most current Linux distributions and Macs are using 2.x as default.

Compatibility Issue: third party packages, modules, libraries, GUI applications, ...

Resources that may help make the choice between Python 2 and Python 3
Python Website: https://wiki.python.org/moin/Python2orPython3

ArcGIS has been upgraded to include the Python 2.7 release since ArcGIS 10.1.
In the latest release of ArcGIS 10.3, ArcGIS Pro uses Python 3.4. This is a significant change from other ArcGIS products including ArcGIS for Desktop, ArcGIS for Server, and ArcGIS Engine that have used (and will continue to use) releases in the Python 2 line.
