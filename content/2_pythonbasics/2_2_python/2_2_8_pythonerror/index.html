Python Error Handling

Back Next

If Python encouters an error with your script, the first step is to determine what type of errors has occurred. This will help you handle the error.

Two types of errors may occur when running your scripts: Syntax Error and Exception.

  • Syntax Errors
  • Exceptions
  • Basic Debugging Tips

Syntax Errors

Syntax refers to the rules used to create classes, objects, and functions, and instructions of your script. Therefore, a syntax error applies to a structural error in which a rule was broken or not enforced.

Common syntax errors

  • Indentation
  • Forgetting to put a colon (:) at the end of statements such as: if, elif, else, for, while, class, def, etc.
  • Wrong comparison operator: using single equal (=) instead of double equals (==).
  • Trying to concatenate a non-string value to a string value.
  • Forgetting a quote to begin or end a string value.
  • A typo for a variable or function or method name (case sensitive).
  • Trying to use a Python keyword (reserved identifiers) for a variable name.

# identation
print "first row"
    print "second row"
	
# missing colon
if greetings == True
    print "Hello!"

# comparison operator
if greetings = True:
    print "Hello!"

# concatenate non-string with string
num_apple = 6
print "I have " + num_apple + " apples."

# missing begining or ending quote
print "I have " + str(num_apple) + apples."

# typo or case-sensitive

Exceptions

After you have corrected all the syntax errors, your script can still fail and generate an error. These errors are referred to as exceptions and are usually caused by errors during the execution of your script.

Exceptions are typically raised because of an error in the logic or decision-making steps of your script. An exception can also occur if your script tries to work with data that it is not expecting. 

>>> l = [1,2,3]
>>> l[3]

Traceback (most recent call last):
  File "", line 1, in 
    l[3]
IndexError: list index out of range

>>> a="2"
>>> pow(a,3)

Traceback (most recent call last):
  File "", line 1, in 
    pow(a,3)
TypeError: unsupported operand type(s) for ** or pow(): 'str' and 'int'
