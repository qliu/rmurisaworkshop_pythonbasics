Python 2 or Python 3?

Python 2.x

Last release is version 2.7 in 2010.

Python 3.x

Pthon 3.0 was released in 2008. After that, 3.x is under active development and has the latest releas of version 3.4 in 2014.

Which version should I use?

3.x downsides: slightly worse library support; most current Linux distributions and Macs are using 2.x as default.

Compatibility Issue: third party packages, modules, libraries, GUI applications, ...
