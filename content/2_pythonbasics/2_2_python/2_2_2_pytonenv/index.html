Choosing Scripting Environments

There is not a specific environment you must use when creating and running a Python script. You can choose the scripting environment that you prefer. However, depending on the task in your script, one environment may be a better choice than the other.

Before getting started to learn Python, you may want to find out which Integrated Development Environments (IDEs) and Text Editors are tailored to make Python editing easy.

List of IDEs
https://wiki.python.org/moin/IntegratedDevelopmentEnvironments

List of Text Editors
https://wiki.python.org/moin/PythonEditors

Python Scripting for ArcGIS: ArcMap Python Window or IDE ?

Short version: Within Python Window, you can quickly test and learn the scripting capabilities and command syntax.
If you want to save those lines of code into a more permanent script on your disk, pass the code to IDLE (Python GUI).
