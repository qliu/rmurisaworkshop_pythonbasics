About Python

Back Next

Python is a widely used general-purpose, object-oriented, high-level programming language.

Readability: Python uses an elegant syntax, making programs you write easier to read.

Cross-platform: Runs on many different computers and operating systems: Windows, MacOS, many brands of Unix, OS/2, ...

Free and Open Source: Python is free software in two senses:
It doesn't cost anything to download or use Python, or to include it in your application.
Python can also be freely modified and re-distributed, because while the language is copyrighted it's available under an open source license.
