Using ArcPy.mapping to Update Texts in the Map

Back Next

ArcPy.mapping

Arcpy.mapping is a Python scripting module that is part of the ArcPy site package.
It was designed primarily to manipulate the contents of existing map documents (.mxd) and layer files (.lyr).
It also provides functions to automate exporting and printing.

Referencing an existing map document.

Typically, one of the first operations you do with an arcpy.mapping script is reference an existing map document (.mxd) or layer file (.lyr) that you want to do something with.

Referencing current map document

>>> mxd = arcpy.mapping.MapDocument("CURRENT")

Change map document properties

>>> mxd.author = "YOUR NAME GOES HERE"

In ArcMap, click File > Map Document Properties. You should see that the author property is set to your name.

Save map document

>>> mxd.save()

Get current map document file path

>>> mxd.filePath

Update texts in the map.

For all the text element on the layout map, replace the typo "Denvre" with "Denver".

>>> mxd = arcpy.mapping.MapDocument("CURRENT")
>>> for elm in arcpy.mapping.ListLayoutElements(mxd,"TEXT_ELEMENT"):
...     elem.text = elm.text.replace("Denvre","Denver")
>>> arcpy.RefreshActiveView()
>>> mxd.save()
